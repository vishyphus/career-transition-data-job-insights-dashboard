# Career Transition & Data Job Insights Dashboard

## Overview

This project is an interactive Power BI dashboard focused on analyzing salaries, career transitions, workplace satisfaction, and hiring-related trends among data professionals.

The dashboard was built using survey data from data professionals across different roles, education levels, industries, and countries. The objective of this project was to transform raw survey data into meaningful business insights through data cleaning, modeling, visualization, and storytelling using Power BI.

---

# Objectives

* Analyze salary trends among data professionals
* Understand the impact of education on salary levels
* Explore how difficult it is to break into data careers
* Identify the most important factors professionals consider while switching jobs
* Visualize relationships between age and salary
* Build an interactive and recruiter-friendly Power BI dashboard

---

# Tools & Technologies Used

* Power BI Desktop
* Power Query
* DAX
* Data Visualization
* Data Cleaning & Transformation

---

# Dashboard Features

## KPI Cards

* Career Switchers %
* Average Salary
* Average Entry Difficulty
* Top Career Priority

## Interactive Visualizations

* Average Salary by Education Level
* Difficulty Breaking Into Data by Education
* Most Important Factors in a New Job
* Age vs Average Salary Scatter Plot

## Interactive Filters

Users can dynamically filter the dashboard by:

* Country
* Job Title
* Education Level
* Industry
* Gender

---

# Key Insights

* Better salary is the top priority among respondents when considering a new job
* Higher education levels generally correlate with higher salary levels
* Over 60% of respondents transitioned into data careers from another field
* Breaking into data is perceived as moderately difficult by many professionals
* Salary trends tend to increase with age and experience
